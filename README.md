# MiBand6_WatchFace

---

小米手环6自制表盘

**请注意：本仓库的表盘仅使用于小米手环6（及其 NFC 版），请不要用于其它型号手环**

[静态图](/静态图) 文件夹下是适用于官方“自定义表盘”的图

[表盘主题](/表盘主题) 文件夹下是本人自制表盘，需要复制到指定目录使用 Zepp Life APP 导入

具体使用方法请看各文件夹下 Readme
